# Voice Assistant Mock Dataset

## Overview

This project contains a structured mock dataset and supporting tools for building, testing, and validating a voice assistant system.

The dataset is designed for:

- Natural Language Understanding (NLU)
- Intent classification
- Entity extraction
- Voice command testing
- Multimodal assistant workflows
- Location-aware AI assistants

The project includes sample data, schemas, documentation, and utility scripts.
