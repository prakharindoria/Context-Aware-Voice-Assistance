# Voice Assistant Intent Documentation

## Overview

This document defines the supported intents for the voice assistant dataset. Intents represent the user's goal or requested action from a voice or multimodal interaction.

Each intent contains:

- Intent name
- Purpose
- Example user queries
- Expected entities
- Assistant behavior
